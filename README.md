# 📊 Telecom Customer Churn Prediction

## 🚀 Project Overview
This project focuses on predicting customer churn in a telecom company using machine learning techniques. The goal is to identify customers who are likely to leave, enabling the business to take proactive retention measures.

---

## 🧠 Problem Statement
Customer churn is a major challenge in the telecom industry. This project aims to:
- Analyze customer behavior  
- Identify churn patterns  
- Build predictive models to classify churn vs non-churn customers  

---

- Matplotlib, Seaborn
